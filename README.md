# OGC Web Coverage Processing Service 1.1 Test-Suite

The OGC Web Coverage Processing Service 1.1 Test-Suite provides the Executable Test Script (ETS) to test implementations against the following specification(s):

  * [WCPS](https://docs.ogc.org/is/08-068r3/08-068r3.html)


Detailed information about this test suite is available [here](http://opengeospatial.github.io/ets-wcps11/).

## Building

This test is build using [Apache Maven](http://maven.apache.org/). To 
build the test suite run maven from the root directory.
```
   % mvn install
```   

Follow the [TEAM ENGINE instructions](http://opengeospatial.github.io/teamengine/installation.html) to install and run the tests in TEAM ENGINE.
